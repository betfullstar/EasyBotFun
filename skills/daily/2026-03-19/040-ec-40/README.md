## 技能简介

基于飞书开放平台API的多维表格自动化操作技能，支持：
1. 自动创建/查询/更新多维表格
2. 数据表管理（创建表、添加字段、查询记录）
3. 视图管理（创建视图、切换展示方式）
4. 记录管理（增删改查、批量操作）

适用于需要自动化处理飞书多维表格数据的场景，如数据同步、报表生成、流程自动化等。
