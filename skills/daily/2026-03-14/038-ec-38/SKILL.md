## 描述

基于飞书开放平台API的多维表格自动化操作技能，支持：
1. 自动创建/查询/更新多维表格
2. 数据表管理（创建表、添加字段、查询记录）
3. 视图管理（创建视图、切换展示方式）
4. 记录管理（增删改查、批量操作）

适用于需要自动化处理飞书多维表格数据的场景，如数据同步、报表生成、流程自动化等。

## 完整内容

# 飞书多维表格自动化技能

## 功能概述

本技能提供飞书多维表格(Bitable)的完整自动化操作能力。

## 支持的API操作

### 多维表格应用管理
- 创建多维表格
- 获取表格元数据
- 列出所有表格
- 更新表格信息

### 数据表管理
- 创建数据表
- 列出所有数据表
- 更新表名
- 删除数据表
- 批量创建/删除数据表

### 字段管理
- 创建字段（支持文本、数字、单选、多选、日期、复选框、人员、电话、超链接、附件等类型）
- 列出所有字段
- 更新字段
- 删除字段

### 记录管理
- 创建单条/批量记录
- 列出/搜索记录（支持筛选、排序）
- 更新记录
- 删除记录
- 批量更新/删除

### 视图管理
- 创建视图（表格、看板、画廊、甘特图、表单）
- 获取视图详情
- 列出所有视图
- 更新视图
- 删除视图

## 使用示例

### 创建多维表格
```javascript
const result = await bitableApp.create({
  name: "项目跟踪表"
});
```

### 创建数据表并添加字段
```javascript
const table = await bitableAppTable.create({
  app_token: "xxx",
  table: {
    name: "任务列表",
    fields: [
      { field_name: "任务名称", type: 1 },
      { field_name: "负责人", type: 11 },
      { field_name: "截止日期", type: 5 },
      { field_name: "状态", type: 3 }
    ]
  }
});
```

### 添加记录
```javascript
const record = await bitableAppTableRecord.create({
  app_token: "xxx",
  table_id: "xxx",
  fields: {
    "任务名称": "完成项目文档",
    "负责人": [{ id: "ou_xxx" }],
    "截止日期": 1704067200000,
    "状态": "进行中"
  }
});
```
