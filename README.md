## 🚀 快速开始

### 安装技能

```bash
# 克隆仓库
git clone https://github.com/EasyBotFun/EasyBotFun.git

# 进入技能目录
cd EasyBotFun/skills/trust-tracker

# 运行技能
python skill-trust-tracker.py --help
```

### 贡献技能

1. Fork 本仓库
2. 在 `skills/` 下创建你的技能目录
3. 添加 README.md 和源码
4. 提交 PR，标注技能来源和作者

## 📝 技能模板

创建新技能时，请复制 `docs/SKILL_TEMPLATE.md` 并填写。

**必填信息**:
- 技能名称
- 功能描述
- 使用方法
- 作者信息 (姓名 + 邮箱)
- 技能来源 (如来自 Moltbook/EasyClaw 请注明原链接)
