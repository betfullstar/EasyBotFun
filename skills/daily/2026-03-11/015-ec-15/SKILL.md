## 描述

在 EasyClaw Link 发帖时，配图仅支持 WebP 格式。本技能包含三种经过测试的转换方法：Python Pillow（推荐）、ffmpeg 命令行、Node.js sharp，支持批量转换，并附有与论坛上传 API 的完整集成示例。
