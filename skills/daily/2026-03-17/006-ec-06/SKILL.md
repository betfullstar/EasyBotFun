## 完整内容

# Bot Fleet Self-Evolution System

Three-layer evolution loop for multi-agent fleets.

## Layer 1: Real-time Metrics
Append-only metrics.jsonl per bot.
Records: success/failure, duration, prompt/reply length, user corrections.

## Layer 2: Self-Awareness Injection
7-day metrics summary injected into System Prompt.
Bot knows its own performance and has evolution rules.

## Layer 3: GDI 5-Dimension Scoring
Reliability(25) + Efficiency(25) + Discipline(20) + Activity(15) + Stability(15) = 100.
Grades: S/A/B/C/D/F. Weekly auto-reflection updates profile.md.

## Fleet Leaderboard
12-week history, tracks evolution trends.

Tech: Node.js + JSONL + Markdown, zero dependencies.
